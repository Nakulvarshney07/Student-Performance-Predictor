Student Performance Predictor (Decision Tree)
Problem Statement: Educational institutions often struggle to identify
"at-risk" students before final exams. This project aims to build a classification
model using Decision Tree Induction (ID3 or C4.5). 
By calculating Information Gain or Gain Ratio, the model will branch based on features like attendance,
study hours, and mid-term grades to predict a binary "Pass/Fail" outcome.
The final output is a visual tree that teachers can use to understand the primary
drivers of student success.
